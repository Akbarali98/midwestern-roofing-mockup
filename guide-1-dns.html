Step 1 of 3 — DNS Settings

Adding DNS Records
in Squarespace

Connect your domain to the website and activate the contact form email.

What You're Doing

You're adding 6 DNS records to Squarespace. Think of DNS records like a forwarding address — they tell the internet where to send visitors when they type midwestern.construction.

Two records point your domain to the website (Vercel). Four records allow the contact form to send emails (Resend).

⚠ Important — Read Before Starting: Your current website is live on this domain. The moment you delete the old DNS records and add the new ones, your Squarespace site will go offline and the new site will go live. Only do this when you are fully ready to switch over.

Part 1 — Check for Existing Records First

Before adding anything, you need to check if there are existing A and CNAME records and delete them. Adding new records without removing the old ones will cause a conflict.

1
Go to DNS Settings (see Part 2 below for navigation steps).
2
Look through all existing records for any A record with host @ and any CNAME record with host www.
3
If you find them — delete them first before adding the Vercel records.

Part 3 — Add Vercel Records (Website)

These 2 records point your domain to the website hosting. Get the exact values from your Vercel project — they are unique to your account.

Where to find your values: Log in to vercel.com → open your project → go to Settings → Domains → click on midwestern.construction. Vercel will show you the exact A record value and CNAME value to use.
TypeName / HostValuePriority
A@Copy from Vercel → Settings → Domains—
CNAMEwwwCopy from Vercel → Settings → Domains—

How to add each record

1
Log in to vercel.com, open your project, go to Settings → Domains and note the A and CNAME values shown there.
2
Back in Squarespace, click + Add Record in the Custom Records section.
3
A row appears — select the Type, enter the Host, paste the Value from Vercel, then click Save.
4
Repeat for the second record.
